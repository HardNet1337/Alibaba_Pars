Инструкция

Файлы экселя создаются автоматически, но не очищаются перед использованием. Перед запуском удалите или очистите их.

Файл Categories содержит ссылки на русский сегмент категорий.
Файл CategoriesFin содержит ссылки на финкский сегмент категорий

Файл Links содержит ссылки на товары, собранные функцией refresh.

Items_log содержит логи об ошибках.

chromedriver.exe всегда должен соотвествовать установленой версии chrome на вашем компьютере. Скачать его можно на официальном сайте или по этой ссылке https://chromedriver.storage.googleapis.com/index.html
Узнать вашу версию chrome можно здесь: настройки > о браузере chrome

.html документы являются хабом для обрабатываемой информации.

_test файлы нужны для коротких тестовых пробегов.

Для корректного полного пробега должны быть установлены следующие параметры:

В функции main(она в самом низу) в параметре refresh установить название файла с ссылками на категории, в зависимости от нужного региона
Так же в этой функции не дожно быть ни одной закомментированной строки. Но если есть ограничения по времени работы,то можно 
сначала закомментить get_link_data и прогнать сначала refresh(Около 2-3 часов), а потом закомментить refresh и раскоментить get_link_data
и сделать прогон уже по ссылкам на товары(Это уже займет очень много времени, сколько точно - предположить не могу)
Если не возможности обработать все ссылки сразу, то можно разбить файл Links.txt на несколько частей и парсить маленькими партиями. 
Так как эксель автоматически не чистится, то результаты таких маленьких прогонов по частям будут дополняться в файл
В функции get_link_data в параметре file_links(по идее должна быть 92 строка) первое значение в кавычках должно быть "Links.txt"
В функции get_href в параметре file_links(по идее 59 строка) такое же значение, как и в функции get_link_data
В функции get_href в параметре times(по идее 33 строка) должно быть значение 100 или больше
